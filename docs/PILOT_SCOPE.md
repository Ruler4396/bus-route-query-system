# PJT-0001 · PILOT_SCOPE

最后更新：2026-03-07  
状态：**首轮试点范围已冻结（A02 完成）**

## 1. 结论
本项目首轮试点不按“广州全城可用”宣传，而是冻结为：

> **广州老城区公共服务走廊试点**  
> 以 **越秀区西部 / 东山片区、荔湾区东部、海珠区西北部** 为主，围绕医院、核心换乘点和公共服务可达性进行验证。

这是一块“已有真实线路和站点基础、后续适合人工核验”的小范围试点，不代表系统已经对广州全城给出可信结论。

---

## 2. 首轮试点区域边界
### 2.1 空间口径
首轮试点聚焦以下连通走廊：
- **越秀东段**：东山署前路 / 东山龟岗 / 中山医 / 烈士陵园 / 农讲所一带
- **越秀西段 / 老城核心**：西门口、人民中路、海珠广场、文化公园一带
- **荔湾东段**：上九路、恩宁路、如意坊、芳村隧道口一带
- **海珠西北段**：凤凰新村、珠江医院、纸厂地铁燕岗站、南石路地铁棣园站一带

### 2.2 为什么选择这一块
1. 当前项目中已有 1 路、3 路、31 路三条真实线路数据经过该走廊。  
2. 这一区域同时覆盖：医院、老城核心换乘点、地铁衔接点、步行风险较高区域，更符合无障碍实验价值。  
3. 相比“全城覆盖”，这一区域更适合作为人工核验和后续真实用户任务验证的首轮样本。

---

## 3. 首轮承诺线路
首轮只把以下线路作为“试点范围内重点保障与优先回归对象”：

### 3.1 1路
- 区间：`东山署前路总站 -> 芳村花园南门总站`
- 关键站点：中山医、烈士陵园、农讲所、海珠广场、市中医院、芳村隧道口、芳村花园南门总站
- 价值：覆盖越秀核心医疗/政务区到芳村居住区的跨江出行场景

### 3.2 3路
- 区间：`如意坊总站 -> 东山龟岗总站`
- 关键站点：广医三院、省中医院、中山图书馆、东华东路、东山龟岗总站
- 价值：覆盖荔湾—越秀之间的医院与公共文化服务场景

### 3.3 31路
- 区间：`解放北路应元路口总站 -> 南石西地铁棣园站总站`
- 关键站点：广医、西门口人民北路、文化公园、珠江医院、纸厂地铁燕岗站、南石路地铁棣园站
- 价值：覆盖老城核心到海珠医疗/地铁换乘场景
